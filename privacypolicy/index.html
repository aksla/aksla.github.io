Privacy policy

Last changed 7th of January 2022

Thanks for using Finn Baby Names (Finn Barnenavn). We have a goal of making it easy, fun and secure for you to find a baby name

Briefly summarized:

  • All data we are storing (except e-mail) are anonymized.
  • The only thing we store about you is your e-mail (you can opt out of this if you use Apple Login). We need to do this to be able to identify you as a user. By doing this we can offer you to log in on other devices, and keep the names that you like.

More information

Storing of data

Storing of data and login in is done by using the Firebase platform from Google. The following data is stored:

  • E-mail (for login)
  • Userdata,for example which names you like. This data is stored anonymously.
In some cases we use the anonymized data to gather statistics of popular names, make suggestions etc. These actions and data can not be traced down to your user. Suggestions are also handled anonymously.

Changes

We reserve the right to change the privacy policy if we see it necessary. Big chances will be announced.
